# Sharli-Pleats-Grace-Business-Website
A modern, responsive business website developed for a local saree pre-pleating entrepreneur. The platform helps customers explore services, view work samples, and book appointments directly through WhatsApp, simplifying customer engagement and reducing manual booking efforts.
Features
1.Responsive mobile-first design
2.Elegant saree-themed UI/UX
3.WhatsApp-integrated appointment booking
4.Service showcase section
5.Video gallery for customer work samples
6.Business contact & social media integration
7.Smooth animations and scroll effects
8.Multi-language support (English & Tamil)

Tech Stack

1.HTML5
2.CSS3
3.JavaScript
4.WhatsApp API Integration

Business Impact

1.Digitized appointment booking process
2.Improved online presence for a local entrepreneur
3.Reduced customer inquiry friction
4.Mobile-friendly experience for wider reach
